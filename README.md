# 2-Player Fighting Game

A simple 2D fighting game built using Python and Pygame where two players can battle in real-time.

## Features
- Player vs Player mechanics
- Attack and hit detection
- Health system
- Keyboard controls (WASD + arrow keys)
- Simple animations
